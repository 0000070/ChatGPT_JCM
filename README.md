# ChatGPT Web
声明：此项目只发布于 Github，基于 Apache2.0 协议，免费且作为开源学习使用。并且不会有任何形式的卖号、付费服务、讨论群、讨论组等行为。谨防受骗。
项目使用Vue2进行开发，给大家提供一个好看的GPT壳子，有好的建议和bug欢迎大家提出来。

![webui2.0](https://i.328888.xyz/2023/03/12/vjVMt.jpeg)

## 下载依赖包
```
npm install
```

## 运行
```
npm run serve
```

## 编译
```
npm run build
```
